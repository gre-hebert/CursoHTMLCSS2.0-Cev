Principais formatações.

1. Negrito

Negrito sem semântica com a tag <b>
Negrito com semântica a tag <strong>

2. Itálico

Itálico sem semântica com a tag <i>
Itálico com semânticacom a tag <em>

3. Marca texto

Texto marcado como a tag <mark>
Texto marcado com outra cor usando o style dentro da tag <mark>

4. Texto grande e pequeno

<Big> para texto grande
<small> para texto pequeno.

5. Texto deletado

Para texto excluido usa-se a tag <del>.

6. Texto inserido

Para texto inserido usa-se a tag <ins>.
A tag <u> sublinha de forma não semântica.

7. Texto sobrescrito e subscrito

<sup> para textos sobrescrito e <sub> para textos subscrito
